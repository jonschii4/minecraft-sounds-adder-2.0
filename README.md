# minecraft-sounds-adder-2.0
Welcome to the *Minecraft Sounds Adder* utility! This tool allows you to easily add completely new, custom sounds to your Minecraft game.

## Key Features
* *Version Compatibility:* Fully supports all versions of Minecraft, including legacy versions like *1.12* up to the latest releases.
* *Streamlined Workflow:* Simplifies the process of integrating custom audio files without manually editing complex JSON config files.
